# OPNet: Bitcoin‑native smart contracts

OPNet brings a secure smart‑contract layer to Bitcoin using **OP‑VM** (a Wasm-based VM) and **AssemblyScript** contracts, with
Bitcoin as the data/settlement layer. This site contains:

- Learn the protocol, consensus, and security model
- Build and test contracts in AssemblyScript
- Run and monitor nodes (Geth‑style docs)
- Explore and try **JSON‑RPC** via an interactive API reference
- Integrate with the **TypeScript SDK** (ethers‑style)

## What makes OPNet different?

- **Bitcoin-first**: transactions/attestations embedded into Bitcoin (witness paths)
- **Deterministic Wasm VM**: AssemblyScript contracts; host syscalls for storage, crypto, calls
- **Unified Accounts**: single key material usable across ecosystems
- **Standards**: OP20 (fungible), OP721 (NFT), access-control, and more
